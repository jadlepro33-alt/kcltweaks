Valorant : +30 FPS sans changer de config

Valorant tourne sur l'Unreal Engine 4 modifié, optimisé pour le compétitif. Mais sans réglages, beaucoup de PCs laissent 30-50% de FPS sur la table. Voici comment maximiser tes FPS sans toucher au matériel — et 100% Vanguard-safe.

Étape 1 : settings vidéo Valorant optimaux

Dans le jeu, va dans Paramètres → Vidéo :

  • Mode d'affichage : Plein écran (jamais Borderless)
  • Résolution : ta résolution native
  • Fréquence d'image max : illimitée (ou +30 FPS au-dessus de ton refresh rate)
  • Qualité du matériel : Faible (tout)
  • Anti-aliasing : MSAA 2x (compromis perf/visibilité)
  • Anisotropic filtering : 2x
  • Texture améliorée : OFF
  • Détail amélioré : OFF
  • UI Vignette : OFF
  • VSync : OFF (active G-Sync à la place côté NVIDIA)
  • NVIDIA Reflex Low Latency : ON + Boost (si GPU NVIDIA)

Ces settings te donnent déjà +20-40% de FPS par rapport au défaut.

Étape 2 : NVIDIA Control Panel pour Valorant

Dans le panneau NVIDIA → Manage 3D Settings → Program Settings → Valorant.exe :

  • Low Latency Mode : Ultra
  • Max Frame Rate : ton refresh rate + 5 (ex: 245 pour écran 240 Hz)
  • Power Management : Prefer Maximum Performance
  • Texture Filtering Quality : High Performance
  • Threaded Optimization : ON
  • Vertical Sync : OFF

Étape 3 : IFEO (Image File Execution Options)

L'IFEO permet de forcer Valorant en priorité High et désactiver Fullscreen Optimizations sans modifier le jeu (donc 100% safe Vanguard). Via registre :

HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\Valorant.exe
PerfOptions (clé) → CpuPriorityClass = 3 (High)

Pour Fullscreen Optimizations Disabled, mets une valeur ~ DISABLEDXMAXIMIZEDWINDOWEDMODE HIGHDPIAWARE dans la compatibilité de Valorant.exe (clic droit propriétés → Compatibilité).

Étape 4 : Vanguard et tweaks safe

Vanguard est très strict. Voici ce qui est safe :

  • Game Mode Windows activé
  • HAGS activé
  • Plan d'alimentation Ultimate Performance
  • System Responsiveness à 0
  • Désactivation Xbox Game Bar + DVR
  • Désactivation Cortana, Phone Link, Phone Companion
  • NVIDIA Reflex / AMD Anti-Lag

Ce qu'il faut éviter :

  • Désactivation VBS / HVCI (Vanguard peut bloquer le lancement)
  • Drivers non signés ou test mode
  • Timer resolution modifié à 0.5ms global
  • Outils de hooking / overlay tiers (RTSS limité)

Étape 5 : services et bloat à éliminer

  • Xbox Live Auth Manager → Manual
  • SysMain (Superfetch) → Disabled (si SSD)
  • Connected User Experiences and Telemetry → Disabled
  • Print Spooler → Disabled (si pas d'imprimante)
  • Windows Search → Manual
  • WaaSMedicSvc → Manual

Étape 6 : config Riot Client en français

Vérifie que ton client Riot est en français, sinon ça consomme un peu plus de bande passante au lancement (téléchargement assets fallback). Paramètres → Langue → Français (France).

Résultats moyens constatés

Sur un setup mid-range (RTX 3060, Ryzen 5 5600X, 16 Go RAM, écran 144 Hz) :

  • Avant : 180-220 FPS en match, drops à 130 sur smokes
  • Après : 270-330 FPS en match, 220+ minimum sur smokes

Sur un setup low-end (GTX 1650, Ryzen 5 2600, 8 Go RAM, écran 60 Hz) :

  • Avant : 80-110 FPS
  • Après : 120-160 FPS
