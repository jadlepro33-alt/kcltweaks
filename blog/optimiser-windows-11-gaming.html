Optimiser Windows 11 pour le gaming en 2026

Windows 11 est devenu le standard, mais il vient avec des paramètres par défaut qui bouffent ressources et FPS. Voici le guide complet 2026 pour récupérer 10 à 20% de performances en gaming.

Pourquoi Windows 11 plombe les FPS

Microsoft a empilé sur Windows 11 plusieurs couches de sécurité (Virtualization Based Security, Memory Integrity, Defender Cloud) qui consomment du CPU même quand tu joues. À ça s'ajoutent des services télémétrie, Cortana, Xbox Game Bar, Phone Link qui tournent en arrière-plan.

Sur un PC gaming dédié, ces fonctions n'apportent rien — et coûtent entre 5 et 15% de FPS dans les jeux compétitifs. La bonne nouvelle : tout est désactivable proprement.

Étape 1 : activer le Game Mode

Le Game Mode de Windows 11 désactive les notifications, met en pause les MAJ et priorise le processus en plein écran. C'est gratuit et 100% safe.

Va dans Paramètres → Jeux → Mode Jeu et active. Si tu as Xbox Game Bar dans la liste, désactive-le aussi (raccourci Win+G inutile et consomme de la RAM).

Étape 2 : Hardware-Accelerated GPU Scheduling (HAGS)

HAGS délègue la gestion mémoire GPU au pilote au lieu de Windows. Sur les GPU récents (NVIDIA 1000+, AMD 5000+) ça réduit la latence et stabilise les FPS.

Direction Paramètres → Système → Affichage → Graphismes → Paramètres graphiques par défaut et active "Planification GPU à accélération matérielle". Redémarre obligatoirement après.

Étape 3 : Memory Integrity (HVCI) à désactiver — avec précautions

Memory Integrity protège le kernel via virtualisation. Coût mesuré : 3 à 10% de FPS sur les CPUs récents. Si tu joues à des jeux compétitifs (Valorant, CS2, Apex), désactive-le.

Attention : ne le désactive PAS si tu utilises ton PC pour du travail sensible (banking, télétravail entreprise). Le compromis sécurité/perf est réel.

Sécurité Windows → Sécurité de l'appareil → Détails de l'isolation du noyau → Intégrité de la mémoire : Désactivé. Redémarre.

Étape 4 : Virtualization Based Security (VBS)

VBS isole le kernel via Hyper-V. C'est l'impact perf le plus élevé : 5 à 15% de FPS sur la plupart des jeux. Pour le couper :

msinfo32

Vérifie la ligne "Sécurité basée sur la virtualisation". Si "En cours d'exécution", elle est active. Pour désactiver via registre :

HKLM\SYSTEM\CurrentControlSet\Control\DeviceGuard
EnableVirtualizationBasedSecurity = 0

Là aussi, redémarre. Vérifie en jeu avant et après : sur certains setups le gain est de 2%, sur d'autres 12%. Ça dépend du CPU.

Étape 5 : services Windows inutiles pour gamer

Liste des services à passer en Manual ou Disabled (selon tes besoins) :

  • SysMain (Superfetch) : préchargement RAM, inutile sur SSD NVMe
  • Windows Search : indexation disque, perte de 5-10% I/O
  • Print Spooler : si tu n'imprimes pas
  • Connected User Experiences and Telemetry (DiagTrack) : télémétrie Microsoft
  • Xbox Live Auth Manager : si tu ne joues pas via Xbox
  • Windows Update Medic Service (WaaSMedic) : force MAJ à des moments incongrus
  • Adobe services : si tu n'utilises pas Creative Cloud actif

Étape 6 : Game Bar + DVR profond

Même désactivé dans Paramètres, Xbox Game Bar laisse des résidus. Pour un kill complet :

HKCU\System\GameConfigStore
GameDVR_Enabled = 0

HKLM\SOFTWARE\Policies\Microsoft\Windows\GameDVR
AllowGameDVR = 0

Désactive aussi BcastDVRUserService dans services.msc.

Étape 7 : System Responsiveness à 0

Windows réserve 20% du CPU pour les tâches background par défaut. Pour les jeux, on veut le contraire : 100% prioritaire au foreground.

HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
SystemResponsiveness = 0

Étape 8 : NVIDIA / AMD / Intel — les bons réglages

Côté GPU, va dans le panneau de contrôle de ton fabricant :

  • NVIDIA : Gérer les paramètres 3D → Mode faible latence = Ultra · Mode de gestion de l'alimentation = Performances maximales
  • AMD : Gaming → Paramètres globaux → Anti-Lag activé · Chill désactivé · Boost désactivé
  • Intel iGPU : Power → Maximum Performance
