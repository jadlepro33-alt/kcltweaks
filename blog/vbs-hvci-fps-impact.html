VBS et HVCI : impact réel sur les FPS

Si tu lis du Reddit gaming, tu as forcément vu : "désactive VBS pour +15% FPS". Vrai ou marketing ? Voici des benchmarks réels sur 5 configs, et la nuance qu'il faut connaître avant de couper la sécurité kernel de ton Windows 11.

Rappel : c'est quoi VBS et HVCI ?

VBS (Virtualization Based Security) utilise Hyper-V pour créer une zone mémoire isolée où Windows stocke ses processus sensibles. Ça empêche les malwares de manipuler ces zones, même en kernel mode. Activé par défaut sur Windows 11 sur les PCs OEM récents.

HVCI (Hypervisor-protected Code Integrity), aussi appelé "Memory Integrity" ou "Core Isolation", est une couche au-dessus de VBS qui vérifie chaque driver kernel avant exécution. Empêche les drivers malicieux ou non signés.

Coût : ces deux fonctionnalités ajoutent une couche d'indirection à chaque appel système → légère perte de performance.

Méthodologie des benchmarks

5 PCs testés, fresh install Windows 11 23H2, mêmes drivers, 1080p ultra settings, 5 runs par jeu, moyenne reportée. Pour chaque PC : test avec VBS+HVCI activés, puis VBS+HVCI désactivés, puis seulement HVCI désactivé.

Setup 3 : Ryzen 7 7800X3D + RX 7900 XT (haut de gamme 2024)

Gain max : 3 à 5% sur tous les jeux. L'impact diminue clairement sur CPU récent avec mitigations matérielles.

Setup 4 : i5-10400F + GTX 1660 Super (CPU 2020 milieu de gamme)

Gain max : 12 à 18% sur tous les jeux. C'est sur ces CPUs plus anciens que l'impact VBS est le plus marqué.

Setup 5 : Ryzen 3 3300X + GTX 1650 (CPU bas de gamme)

Gain max : 15 à 22%. CPU déjà à la peine, chaque cycle économisé compte.

Conclusions

1. L'impact dépend ENORMEMENT du CPU

Si tu as un Ryzen 7800X3D ou un i7-13700K+, ne te casse pas la tête : 3-5% de gain ne valent pas la sécurité perdue. Garde VBS et HVCI activés.

Si tu as un CPU plus ancien (3rd-5th gen Intel, Ryzen 1xxx/2xxx, ou tout CPU avant 2020), désactiver VBS+HVCI peut te donner 10 à 22% de FPS. Le compromis devient intéressant.

2. Le compromis sécurité

Désactiver VBS et HVCI rend ton Windows plus vulnérable à certains exploits (drivers vulnérables, escalation kernel). Si tu utilises ton PC uniquement pour gaming et navigation occasionnelle, le risque réel est très faible. Si tu fais du télétravail entreprise, banking sensible, ou stockes des secrets, garde-les activés.

3. Compatibilité anti-cheat

Vanguard (Valorant) peut refuser de lancer le jeu sur Windows 11 si HVCI est désactivé. Test ton compte secondaire d'abord. FACEIT AC est encore plus strict. EAC (Apex, Fortnite, R6) est plus tolérant.

4. HVCI seul plutôt que tout désactiver

Compromis intéressant : désactive juste HVCI (Memory Integrity) et garde VBS. Tu récupères environ 60% du gain perf tout en gardant l'isolation kernel principale. C'est ce que recommande Microsoft pour les gamers.

Comment vérifier l'état VBS et HVCI

msinfo32

Cherche les lignes :

  • "Virtualization-based security" : Running = VBS actif
  • "Virtualization-based security Services Running" : si contient "Hypervisor enforced Code Integrity" = HVCI actif

Comment désactiver proprement

HVCI seul : Sécurité Windows → Sécurité de l'appareil → Isolation du noyau → Intégrité de la mémoire : Désactivé. Redémarre.

VBS complet : via registre

HKLM\SYSTEM\CurrentControlSet\Control\DeviceGuard
EnableVirtualizationBasedSecurity = 0
RequirePlatformSecurityFeatures = 0

Reboot obligatoire. Vérifie avec msinfo32.
